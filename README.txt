# JCrud

##### A simple structure for the best use of the concept of crud with java
---
>This project aims to create a connection to the SQLite database using JDBC.
The ultimate goal is to create a project with a structure that allows for better enjoy the use of the concept of CRUD.

## Requirements
- Dependency JDBC installed

## Characteristics:

- Syntax easy to understand and use;
- Clean code;
